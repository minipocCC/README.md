# NovaFlow

## Resumen Ejecutivo

### Descripción
NovaFlow es una aplicación diseñada para gestionar y automatizar procesos básicos de manera eficiente. Su objetivo es facilitar tareas que normalmente requieren tiempo, coordinación y organización manual.

### Problema identificado
Los usuarios no contaban con una herramienta clara y accesible para realizar estas tareas de forma ordenada, lo que generaba errores, retrasos y pérdida de información.

### Solución
NovaFlow centraliza las funciones principales, automatiza procesos y mejora la experiencia tanto del usuario final como del administrador, reduciendo tiempos y aumentando la precisión operativa.

### Arquitectura
Arquitectura simple basada en:
- Frontend (si aplica)
- Backend (Java)
- Base de datos (H2 o PostgreSQL)
- Servidor embebido (Spring Boot)
